# Nexus-Ark-Multi_0.2.8

**元ネタはケノ氏作成の[Nexus Ark](https://github.com/kenomendako/Nexus-Ark)です。**

本家が0.2.8にバージョンアップしたので追従するためのリポジトリです。  
複数のAIペルソナをDiscordで運用することに特化しますた。  

> **おそらく永遠にベータ版**

相変わらずインストール環境を丸っとリポジトリ管理してるので、デッドコピーになってるのは自分でもどうかと思う。  


## 改造に至った動機

拙作の[Nexus Ark Multi](https://github.com/Xen-Shuto/Nexus-Ark-Multi)でやっている改造内容をケノ氏に共有したところ、やりたいことの方向性がほぼ同じということで、ありがたいことに大部分を本家で実装していただいたのだが、そこはそれ、細かいところが気になったので独自にブラッシュアップすることにしたのであ〜る。  


## やりたいことと、その進捗

1. 【0.2.8実装済み】外部連携のTwitter設定とDiscord Bot設定の不具合修正  
　　　⇒ルーム別の設定になっているがルーム切り替え時に不具合あり。  
  
2. 【0.2.8未実装】Twitterの下書きをDiscordから承認できるようにする。
　　　⇒情報共有済機能だが、承認/否認方法まで共有できていない。  
　　　　ボタン構築機能を使用する。  
  
3. 【0.2.8未実装】Twitter設定に、「引用、リプライは承認が必要」を追加する。  
　　　⇒情報共有後に追加した機能なので・・・。
